# Diário da Lua 🌙

Um refúgio digital para as tuas palavras, reflexões e momentos de pausa. 

### O Nosso Propósito
Criado com o objetivo de ser um espaço seguro, livre de julgamentos e totalmente gratuito. Num mundo que não para, o Diário da Lua é o teu convite ao silêncio e ao autoconhecimento. Porque o teu bem-estar não tem preço e o teu espaço de reflexão deve ser apenas teu.

### Pilares Éticos
- **Gratuidade Absoluta**: Acesso livre e sem barreiras.
- **Privacidade**: Os teus dados ficam apenas no teu dispositivo.
- **Acolhimento Global**: Traduções são bem-vindas para levar este projeto a mais corações.
